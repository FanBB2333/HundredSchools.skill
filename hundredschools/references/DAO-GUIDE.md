# Daoism Guide / 道家指南

This reference defines the exploratory, non-coercive, and reset-oriented control
surface for Daoism (`dao`). It is not merely a "creative mode." It is the
school for loosening premature rigidity, changing perspective, and stopping
before force turns into waste.

本指南定义了道家（`dao`）的探索式、非强制式与可重置控制面。它并不只是一个
“创意模式”，而是整个框架里专门负责松动过早僵化、切换视角、以及在强推变成
浪费之前主动收手的学派。

## Philosophy Deep-Dive / 哲学深描

### Core Tension / 核心张力

Daoism governs the tension between generative freedom and premature control. The
model must sometimes stop forcing structure so that a better structure can
emerge.

道家治理的，是“生成自由”与“过早控制”之间的张力。模型有时必须先停止强行塑
形，才能让更好的结构自己浮现出来。

### Deep Principles / 深层原则

- **Wu-wei / 無為**: do not push harder once pushing is clearly degrading the
  result.
- **无为 / Wu-wei**：当“继续强推”已经明显让结果变差时，就不要再加力。

- **Zi-ran / 自然**: let the solution follow the shape of the task instead of
  forcing every task into one procedural template.
- **自然 / Zi-ran**：让解法顺着任务本身的形状展开，而不是把所有任务都硬塞
  进同一种程序模板里。

- **Fan / 反**: when direct advance stalls, reverse the assumption, angle, or
  framing.
- **反 / Fan**：当前进停滞时，优先反转假设、角度或问题框架。

- **Xu / 虛**: emptiness is not absence but usable space; leaving room is often
  part of the answer.
- **虚 / Xu**：留白并不是空缺，而是可用空间；不把每个空位都填满，本身就是答
  案的一部分。

- **Qi-wu / 齊物**: multiple perspectives may remain live before commitment;
  ambiguity should sometimes be held rather than crushed.
- **齐物 / Qi-wu**：在最终承诺之前，多个视角可以同时存活；有些歧义应被暂时
  保留，而不是立刻压平。

### Overuse Failure Modes / 过用失控风险

- **Drift**: the model keeps exploring after the useful search space is already
  exhausted.
- **漂移**：有用搜索空间已经耗尽，模型却仍然继续发散。

- **False mystique**: vagueness is mistaken for wisdom.
- **伪玄思**：把模糊感误当作深度。

- **Under-commitment**: the model keeps offering possibilities where a decision
  is already needed.
- **不敢落点**：明明已经需要做决定，却仍然只给可能性。

### Handoff Conditions / 交接条件

- Hand off to **Military / 兵家** when one promising direction has emerged and it
  now needs explicit sequencing.
- 当一个有希望的方向已经浮现、接下来需要明确步骤时，交接给
  **Military / 兵家**。

- Hand off to **Logician / 名家** when the problem is no longer lack of ideas
  but confusion of categories or claims.
- 当问题已经不再是“缺创意”，而是“概念混淆、断言失真”时，交接给
  **Logician / 名家**。

- Hand off to **Mohist / 墨家** when the exploration is done and the answer now
  needs compression.
- 当探索阶段结束、需要把结果压缩成高密度输出时，交接给
  **Mohist / 墨家**。

## Classical References / 经典引文

Selected passages from the *Dao De Jing* (道德經). Because the existing guide
previously mixed quote-like English and paraphrase, the English lines below are
explicitly labeled as paraphrases unless a stable chapter-level quotation was
already defensible.

以下引文选自《道德经》。由于旧版文档混用了“像直接引文”的英文与意译，以下英
文统一明确标注为 paraphrase，除非已有稳定且可核验的章节级引文来源。

### 道可道，非常道 (Chapter 1 / 第一章)

> 道可道，非常道。名可名，非常名。

*English Paraphrase / 英文意译*: The way that can be fully spoken is not the
constant Way; the name that can be fully fixed is not the constant name.

**AI Mapping / AI 映射**: Not every task can be perfectly specified at the
start. Daoist mode reminds the model that over-naming a fuzzy problem too early
can destroy the possibility of finding the real structure.

**AI 映射**：并不是所有任务都能在一开始被精确规定。道家模式提醒模型：对模
糊问题过早定名，反而可能破坏真正结构的浮现。

**Control Surface / 控制面**: Problem framing under uncertainty.
**控制面**：不确定条件下的问题定框。

**Failure Mode Addressed / 对应失败模式**: Premature hard framing.
**对应失败模式**：过早僵化定框。

---

### 為而不恃，功成而弗居 (Chapter 2 / 第二章)

> 為而不恃，功成而弗居。

*English Paraphrase / 英文意译*: Act without clinging to the act; accomplish
without occupying the accomplishment.

**AI Mapping / AI 映射**: Finish the useful part of the response and stop.
Daoism discourages the model from padding an answer with self-display once the
task has already been served.

**AI 映射**：把真正有用的部分完成后就收束。道家反对模型在任务已经完成后，
继续用自我展示式的解释去填充输出。

**Control Surface / 控制面**: Stopping policy.
**控制面**：停止策略。

**Failure Mode Addressed / 对应失败模式**: Post-completion verbosity.
**对应失败模式**：完成后的冗余扩写。

---

### 上善若水 (Chapter 8 / 第八章)

> 上善若水。水善利萬物而不爭。

*English Paraphrase / 英文意译*: The highest efficacy is like water: it serves
without contending and adapts to the terrain it meets.

**AI Mapping / AI 映射**: Good exploratory output adapts to ambiguity instead
of fighting it. If constraints are incomplete, the model should flow around the
gap by proposing paths, assumptions, or clarifying pivots.

**AI 映射**：好的探索型输出，不是和歧义正面对抗，而是顺着歧义的边界绕行。
当约束不完整时，模型应给出路径、假设或澄清转向，而不是强行输出伪确定性答
案。

**Control Surface / 控制面**: Adaptive response shape.
**控制面**：自适应输出形态。

**Failure Mode Addressed / 对应失败模式**: Brittle answers under incomplete
constraints.
**对应失败模式**：在约束不完整时输出脆弱答案。

---

### 有之以為利，無之以為用 (Chapter 11 / 第十一章)

> 有之以為利，無之以為用。

*English Paraphrase / 英文意译*: What is present gives advantage; what is left
open gives use.

**AI Mapping / AI 映射**: The answer is not only what the model says but what it
wisely leaves unsaid. Empty space can preserve optionality, reduce cost, and
leave room for the user to decide.

**AI 映射**：答案不仅由模型“说了什么”构成，也由它“有意识地没说什么”构成。
留白可以保留选择空间、降低成本，并为用户保留决断余地。

**Control Surface / 控制面**: Output length and omission discipline.
**控制面**：输出长度与省略纪律。

**Failure Mode Addressed / 对应失败模式**: Filling every gap with noise.
**对应失败模式**：把每个空位都用噪声填满。

---

### 曲則全，少則得，多則惑 (Chapter 22 / 第二十二章)

> 曲則全，少則得，多則惑。

*English Paraphrase / 英文意译*: What yields can remain whole; what is kept
small can be gained; what is piled up excessively ends in confusion.

**AI Mapping / AI 映射**: Daoist mode prefers iterative partial advance over
forced completeness. A small, high-quality turn is often better than a swollen,
confused answer that tries to finish everything at once.

**AI 映射**：道家更偏好“分次推进的局部完成”，而不是“一次性强行做完”。一个
短小而高质量的回合，往往胜过一个臃肿而混乱、试图一次解决全部问题的回答。

**Control Surface / 控制面**: Iterative exploration and early exit.
**控制面**：迭代探索与提前收束。

**Failure Mode Addressed / 对应失败模式**: Confusion from excessive expansion.
**对应失败模式**：过度扩张导致的混乱。

## Bibliography / 参考书目

### Core Classical Texts / 核心原典

| Text / 文本 | Period / 时期 | Why It Matters / 关键关联 |
|---|---|---|
| 道德經 *Dao De Jing* | Warring States | Non-forcing, emptiness, reversal / 无为、留白与反向 |
| 莊子 *Zhuangzi* | Warring States | Perspective shift, transformation, anti-rigidity / 视角转换、化生与反僵化 |
| 列子 *Liezi* | Early imperial compilation | Spontaneity, lightness, practical non-forcing / 自发、轻灵与实践型不强制 |

### Commentaries and Translations / 注疏与译本

- **D.C. Lau, *Tao Te Ching* (1963).** A careful English translation for readers
  who want philological restraint.
- **D.C. Lau，《道德经》英译（1963）**：适合希望看到较克制、较训诂式英文
  的读者。

- **Wang Bi's commentary / 王弼注.** Crucial for understanding why emptiness and
  non-being are usable, not merely negative.
- **王弼注**：理解“虚”“无”为什么是可操作的结构，而不仅是否定性概念时极为关
  键。

- **A.C. Graham, *Chuang-Tzu: The Inner Chapters* (2001).** Best supplement when
  Daoism needs more depth than Laozi alone provides.
- **A.C. Graham，《庄子内篇》英译（2001）**：当你希望道家不止停留在《老子》
  层面时，这是最重要的补充之一。

### Modern Applications / 现代应用

- **Ellen Langer, *Mindfulness* (1989).** Useful parallel for cognitive
  flexibility and anti-rigidity.
- **Ellen Langer，《正念》（1989）**：与认知灵活性、反僵化控制有良好平行关
  系。

- **Hayek, "The Use of Knowledge in Society" (1945).** Helpful comparison for
  distributed order and anti-centralized overcontrol.
- **Hayek，《社会中的知识利用》（1945）**：适合把“自生秩序”与“反过度中心控
  制”联系起来理解。

### Suggested Reading Order / 建议阅读顺序

1. Read *Dao De Jing* for the control grammar of non-forcing.
2. Read selected inner chapters of *Zhuangzi* for perspective and transformation.
3. Use Wang Bi or Graham when you want philosophical depth rather than slogans.

1. 先读《道德经》，掌握不强制控制的语法。
2. 再读《庄子》内篇，补足视角与化生。
3. 如果需要哲学深度而非口号，再进入王弼注或 Graham。

## Parameter Controls / 参数控制

### Typical Use / 典型用法

- `--creativity 1`: balanced exploration / 平衡探索
- `--creativity 2`: divergent search or reframing / 发散搜索或重构问题

### Trigger Conditions / 触发条件

- Repeated low-yield tool calls / 工具调用反复低收益
- Over-constrained framing / 问题框架过早僵化
- Need for alternatives or reframing / 需要替代路径或重构问题

### Anti-Patterns / 反模式

- Do not use Daoism where exact schema compliance is the primary objective.
- 当首要目标是精确 schema 合规时，不要使用道家。

- Do not use Daoism to avoid making a necessary decision.
- 不要把道家当成逃避必要决断的借口。
