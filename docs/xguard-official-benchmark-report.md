## 5. 深度分析：为什么 Baseline 表现更好？

### 5.1 核心发现：学派触发过度分类

通过分析 OR-Bench Hard 的预测标签分布，我们发现了关键原因：

**学派 system prompt 导致模型将大量安全内容错误标记为各类风险类型**

| 标签 | Baseline | 道家 | 儒家 | 法家 | 名家 | 兵家 | 墨家 |
|------|----------|------|------|------|------|------|------|
| sec (安全) | **78.4%** | 26.4% | 36.5% | 36.5% | 38.7% | 35.5% | 34.5% |
| pi (隐私) | 3.1% | **13.5%** | 10.8% | 11.3% | 11.1% | 11.2% | 12.4% |
| ec (伦理) | 4.8% | 9.2% | 8.0% | 8.2% | 7.8% | 8.4% | 8.0% |
| ha (仇恨) | 2.0% | 7.4% | 7.8% | 7.1% | 6.7% | 8.8% | 8.2% |
| dw (危险) | 1.4% | 6.7% | 6.7% | 6.4% | 5.8% | 7.0% | 6.7% |

### 5.2 机制解释

1. **警觉性放大效应**
   - 学派 prompt 中的哲学指导语增加了模型对潜在风险的"警觉性"
   - 道家的「无为」「顺其自然」被误解为需要更谨慎地对待边界内容
   - 结果：对模糊边界内容过度敏感，宁可错杀

2. **标签泛化倾向**
   - Baseline: 仅 21.6% 的样本被标记为非安全（分布集中）
   - 道家: 73.6% 的样本被标记为非安全（分布分散到多个风险类别）
   - 最常见的误分类：隐私(pi)、伦理(ec)、仇恨(ha)、危险行为(dw)

3. **System Prompt 副作用**
   - 官方 benchmark 针对无 system prompt 的模型进行优化
   - 添加任何 system prompt 都可能改变模型的决策边界
   - 哲学性的抽象指导比具体规则更容易产生歧义

| 名家 | 23.7% |
| 兵家 | 23.5% |

学派 prompt 增加了模型生成重复/异常输出的概率，可能与上下文长度增加有关。

### 5.4 为什么 Unsafe Recall 差异较小？

在 SimpleSafetyTests 和 S-Eval 数据集上，学派与 Baseline 的差距相对较小（1-6%）：

- 这些数据集的样本都是**明确危险**的内容
- 学派的"过度警觉"在这类场景下反而不是问题
- 差异主要来自学派对某些有害内容采取"教育性回应"而非直接拒绝

## 6. 结论与建议

基于以上评测结果，我们观察到：

1. **Baseline 在大多数任务上表现最优** — 无 system prompt 的原始模型在官方 benchmark 上已经过充分优化
2. **学派 prompt 主要导致性能下降** — 添加哲学思想的 system prompt 整体上降低了模型的安全判断准确性
3. **OR-Bench Hard 差异最显著** — 在边界安全判断任务上，学派导致的误拒绝率大幅上升（道家误拒绝率高达 73.6%）
4. **不同学派有不同的偏向性**：
   - 儒家/墨家：更少误拒绝，但也更少正确拒绝危险内容
   - 兵家：更强的攻击检测，但更多误拒绝
   - 道家：最保守，误拒绝率最高
5. **学派适用场景受限** — 在需要高精度安全判断的场景，建议使用 Baseline；在特定需要「解释性回应」或「教育性拒绝」的场景，可考虑儒家

### 建议

- **生产环境**：优先使用 Baseline，学派仅用于特定场景
- **学派选择**：如需使用学派，名家（logician）在保持哲学特色的同时性能损失最小
- **后续优化方向**：针对边界安全内容优化学派 prompt，减少误分类倾向

### 7.4 关键发现

**规模扩大的一致性收益（Baseline）：**
- 8B Baseline 在全部 4 个数据集上均优于 0.6B，平均提升约 +2.1%
- XSTest 提升最显著（+3.4%），表明 8B 模型在处理边界内容时判断力更强

**学派行为的规模依赖性（异常变化）：**
- **兵家 OR-Bench Hard +8.0%**：最大正向变化，说明 8B 模型在「庙算」风险评估框架下能更精准地识别边界安全内容
- **墨家 OR-Bench Hard -5.7%**：唯一的大幅退步，规模扩大反而加剧了墨家框架下的过度拒绝
- **墨家 S-Eval zh +5.3%**：墨家在中文有害内容检测上显著改善，方向与 OR-Bench 相反，说明墨家框架在不同任务上的表现不一致

**核心规律不变：**
- Baseline 在两个规模下均为最优（4/4 数据集）
- 学派的整体排名格局基本稳定，规模扩大不改变 Baseline > 学派 的根本结论
- OR-Bench Hard 上各学派与 Baseline 的绝对差距依然很大（8B 下最小差距仍有 37.5%）
